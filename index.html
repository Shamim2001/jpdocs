How to login and log out

How to Login

Visit https://jp.pixcafe.xyz/register

  • Input your login details
  • Click on the login button
  • You will see the homepage if login success. (if not it will show an error)
  • To access the dashboard hover on the top bar and click on dashboard.
  • You’ll see the dashboard now.

How to log out

  • If you’re on the dashboard (admin) you can log out by hovering the name on the top right side and you’ll see the logout option.

    How to add event page

    • Go To Dashboard
    • Click Events Tab
    • Find page Add New button Click Add New Button
    • You’ll see a new event form where you should put the title, Content, Pusblish and Thumbnail
    • You have to fulfill those input field by below instruction
    • Content: Type event content in content input field.
      • link-input: You can type link url, text, title of your link. And can also choose, should this link open in this tab or in a new tab?
        Image-icon: You can add img in content by click link img icon content top bar.
        Image-input: You can type img source (from another website), alternative description (if img is not found this text will show), image height and width, title of your link. And can also choose, to show or hide the photo caption?
        Image-input by upload: You can upload img from image module "Upload" tab.
    • Publish Options: You get 3 type publish option.
      Publish: If you select this, everyone will see this event on the frontend.
      Private: If you select this, only you will see this event on the frontend.
      Draft: If you select this, no one on the frontend will see this event. And it will be saved in your event list draft.
    • Thumbnail: You can upload event photos from here.
    • Category: You can select your event category from here.
      Category: create an category by Add New Category click this button.
    • Watch the video to understand well.
    video
